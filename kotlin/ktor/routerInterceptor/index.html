# ktor 自定义验证拦截器

# 先来看看官方的「authenticate」源码

  1. 可以发现官方直接注册一个 Authentication 模式并拦截请求管道实现校验。

# 自行实现

  3. 随后我们通过 createChild() 获取一个子 Route 类。
