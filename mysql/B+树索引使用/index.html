# InnoDB 头记录信息

名称作用
record_type表示记录的类型
next_record表示记录下一跳记录
field_info字段信息
其他信息表示记录的其他的信息

record_type 的取值代表下面的意思

  • 0: 普通的用户记录
  • 1: 目录记录
  • 2: Infimum 记录
  • 3: Superemum 记录
